# Netflix Content Strategy & Trend Analysis (Python)

## 📌 Project Objective
To analyze Netflix content data and extract insights about content distribution, country contribution, ratings, and release trends.

## 🛠 Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Key Analysis Performed
- Data cleaning and handling missing values
- Movies vs TV Shows comparison
- Country-wise content distribution
- Content rating analysis
- Release year vs Date added trends
- Top genres and production insights

## 📊 Key Insights
- Majority of content was added after 2015
- Movies dominate over TV Shows
- United States contributes highest content volume
- Netflix content focuses largely on mature audience ratings

## 💡 Business Value
These insights can support:
- Content acquisition strategy
- Market expansion decisions
- Genre-focused production planning
